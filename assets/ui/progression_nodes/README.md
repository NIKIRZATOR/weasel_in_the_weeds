# Иконки узлов прогрессии

В этой папке лежат изображения для дерева развития героя.

## Рекомендации

- используйте `png` с прозрачным фоном;
- лучше держать квадратное соотношение сторон;
- удобные размеры: `64x64` или `128x128`.
