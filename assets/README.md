# Каталог `assets`

Здесь находятся игровые ресурсы: графика, интерфейс, портреты, музыка и звуки.

## Основные подпапки

- `audio/` - музыка, эмбиент и звуковые эффекты.
- `bosses/` - графика и портреты боссов.
- `characters/` - спрайты героя.
- `enemies/` - спрайты обычных врагов.
- `items/` - иконки предметов.
- `npc/` - NPC и их портреты.
- `system/` - фоновые изображения и системные элементы интерфейса.
- `ui/` - иконки интерфейса и прогрессии.
- `world_objects/` - объекты мира и окружение.

## Рекомендации по ассетам

- Используйте `png` с прозрачностью для спрайтов и иконок.
- Для UI придерживайтесь единого стиля и размера.
- Пути в игровых каталогах обычно указываются относительно `assets/`.

## Где настраиваются ссылки

- Предметы: `game/items/catalog_data.json`
- Ресурсы мира: `game/objects/*.json`
- Локализация интерфейса: `game/localization/translations/*.json`
