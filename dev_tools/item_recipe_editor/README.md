# Редактор предметов и рецептов

Файл запуска:

```text
dev_tools/item_recipe_editor/item_recipe_editor.py
```

## Запуск

```bash
python dev_tools/item_recipe_editor/item_recipe_editor.py
```

## С какими файлами работает

- `game/items/catalog_data.json`
- `game/crafting/recipes_data.json`

## Возможности

- создавать и редактировать предметы;
- управлять иконками, описаниями и параметрами;
- создавать и редактировать рецепты;
- настраивать ингредиенты, результат и стоимость открытия;
- дублировать и удалять записи;
- сохранять изменения обратно в JSON.
