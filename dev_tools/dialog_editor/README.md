# Редактор диалогов

Файл запуска:

```text
dev_tools/dialog_editor/dialogue_editor.py
```

## Запуск

```bash
python dev_tools/dialog_editor/dialogue_editor.py
```

## Что умеет

- создавать новые диалоги;
- редактировать узлы и переходы;
- настраивать говорящего и портрет;
- добавлять варианты ответа;
- выдавать награды и сюжетные флаги;
- проверять целостность ссылок между узлами.

## Где сохраняются данные

Диалоги сохраняются в папку:

```text
dialogues/
```
