# Map Object Editor

Редактор размещает игровые объекты поверх готовой карты уровня.

## Запуск

```bash
python dev_tools/map_object_editor/map_object_editor.py
```

## Что редактирует

- для TMX-уровней обновляет `levels/<level>/objects.json`;
- для старых JSON-уровней обновляет список `objects` внутри файла уровня.

## Базовый сценарий работы

1. Выбрать уровень.
2. Выбрать объект в палитре.
3. Поставить объект на карту.
4. При необходимости отредактировать свойства.
5. Нажать `Apply`, затем `Save`.

## Управление

- ЛКМ по пустой клетке - поставить объект.
- ЛКМ по объекту - выбрать объект.
- ПКМ по объекту - удалить объект.
- `Duplicate` - создать копию рядом.
- `Save` - сохранить изменения.
