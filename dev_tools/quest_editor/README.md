# Редактор квестов

Файл запуска:

```text
dev_tools/quest_editor/quest_editor.py
```

## Запуск

```bash
python dev_tools/quest_editor/quest_editor.py
```

## Что умеет

- редактировать квесты отдельно для каждого уровня;
- настраивать цели, категории и порядок отображения;
- задавать обязательные сюжетные флаги;
- настраивать диалог активации;
- сохранять квесты в `levels/<level>/quests.json`.

## Поддерживаемые типы целей

- `event`
- `flag`
- `item`
