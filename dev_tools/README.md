# Каталог `dev_tools`

Здесь лежат вспомогательные инструменты разработки.

## Сейчас доступны

- `dialog_editor/` — редактор диалогов
- `item_recipe_editor/` — редактор предметов и рецептов

## Запуск

Запускать инструменты лучше из корня проекта.

Примеры:

```bash
python dev_tools/dialog_editor/dialogue_editor.py
python dev_tools/item_recipe_editor/item_recipe_editor.py
```

## Назначение

Эти инструменты нужны, чтобы не редактировать большие JSON-файлы руками:

- диалоги;
- предметы;
- рецепты.
