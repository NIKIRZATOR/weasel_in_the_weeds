# Каталог `game/objects`

Здесь собраны объекты мира, с которыми сталкивается игрок.

## Основные классы

- `world_object.py` — базовый объект мира.
- `solid_object.py` — твердый объект, который блокирует движение.
- `interactable_object.py` — объект, с которым можно взаимодействовать.
- `pickable_object.py` — объект, который можно поднять.
- `gatherable_object.py` — объект-сборщик ресурсов.
- `checkpoint_object.py` — чекпоинт/камень воскрешения.
- `grass_hide_zone.py` — трава, в которой можно скрыться.
- `level_transition.py` — переход между уровнями.
- `npc_object.py` — NPC с диалогом.
- `factory.py` — фабрика создания объектов из JSON.

## Каталоги и справочные файлы

- `gatherable_catalog.json` — шаблоны gatherable-объектов.
- `gatherable_catalog.py` — загрузка шаблонов.
- `about_objects.md` — справка по формату объектов.
- `about_npc_dialogues.md` — справка по диалогам NPC.

## Как создаются объекты

1. Уровень загружается из JSON.
2. Каждый объект из списка `objects` передается в `factory.py`.
3. Фабрика выбирает нужный класс и создает экземпляр.
4. `GameScene` обновляет и отрисовывает эти объекты.

## Важные взаимодействия

- `checkpoint_object` меняет точку возрождения игрока;
- `grass_hide_zone` делает игрока скрытым;
- `level_transition` переносит игрока на другую локацию;
- `npc_object` открывает `DialogueScene`;
- `gatherable_object` выдает ресурсы и может выдавать `knowledge_shards`.

## Подписи объектов

Пока нет текстур, базовый `world_object.py` умеет рисовать название объекта внутри его области.
