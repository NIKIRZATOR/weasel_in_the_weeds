# Структура каталога `game`

Этот каталог содержит весь основной код игры.

## Что здесь лежит

- `core` — базовые утилиты: камера, векторы, таймеры, загрузка ассетов.
- `crafting` — система рецептов и крафта.
- `dialogues` — загрузка диалоговых JSON-файлов.
- `entities` — игрок и враги.
- `items` — предметы, инвентарь, экипировка, каталог предметов.
- `objects` — интерактивные и статичные объекты мира.
- `scenes` — сцены игры: меню, игровой мир, инвентарь, карта, диалоги, крафт.
- `ui` — HUD и поверхностный интерфейс.
- `world` — загрузка уровней, тайловая карта, коллизии.

## Как все связано

1. `GameApp` из `scenes/app.py` запускает текущую сцену.
2. Игровая сцена `GameScene` загружает уровень через `world/level.py`.
3. На основе JSON-уровня создаются:
   - тайловая карта,
   - объекты мира,
   - враги,
   - игрок.
4. Игрок взаимодействует с объектами, собирает предметы, открывает карту, инвентарь, диалоги и крафт.
5. UI из `ui/hud.py` отрисовывает полоски, хотбар, монеты, осколки знаний и подсказки.

## Что менять в первую очередь

- Новая механика мира: обычно `objects`, `entities`, `world`, `scenes/game_scene.py`
- Новый предмет: `items/catalog_data.json`
- Новый рецепт: `crafting/recipes_data.json`
- Новый диалог: каталог `../dialogues`
- Новый уровень: каталог `../levels`
