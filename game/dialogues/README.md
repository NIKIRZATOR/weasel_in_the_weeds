# Каталог `game/dialogues`

Это служебный пакет для загрузки диалоговых файлов.

## Файлы

- `loader.py` — функция загрузки JSON-диалога по пути.
- `__init__.py` — экспорт загрузчика.

## Как это используется

1. Объект NPC указывает файл диалога.
2. Загрузчик читает JSON из каталога диалогов.
3. Сцена `DialogueScene` показывает узлы, выборы и награды.

## Что здесь не хранится

Сами данные реплик лежат не здесь, а в корневом каталоге `dialogues/`.
