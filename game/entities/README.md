# Каталог `game/entities`

Здесь находятся живые сущности игры: игрок и враги.

## Файлы

- `entity.py` — базовый класс сущности.
- `player.py` — логика игрока.
- `enemy.py` — логика врагов ближнего и дальнего боя.

## Игрок

Игрок хранит:

- здоровье и выносливость;
- инвентарь, хотбар, экипировку;
- сюжетные флаги;
- открытые участки карты;
- полученные награды из диалогов;
- `knowledge_shards`;
- открытые рецепты.

## Враги

Враги обновляются каждый кадр в `GameScene`, двигаются по своим правилам и получают урон от атаки игрока.

## Когда менять

- новая боевая механика;
- новые параметры игрока;
- новая логика ИИ врага;
- новая постоянная прогрессия игрока.
