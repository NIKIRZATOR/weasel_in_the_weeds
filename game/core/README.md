# Каталог `game/core`

Здесь лежат низкоуровневые вспомогательные классы, которые используются по всему проекту.

## Файлы

- `assets.py` — загрузка изображений и подготовка графики.
- `camera.py` — камера, которая следит за игроком и ограничивается размерами мира.
- `timer.py` — простой таймер для прыжка, атаки, рывка, получения урона и других временных состояний.
- `vector.py` — векторная математика для движения, направления и смещений.

## Роль в проекте

Эти модули не знают про конкретные сцены, NPC или рецепты. Они дают базовые инструменты, на которых строится остальная игра.

## Когда сюда лезть

- если нужно улучшить камеру;
- если нужна новая временная механика;
- если нужно изменить загрузку спрайтов;
- если нужно добавить общее математическое поведение.
