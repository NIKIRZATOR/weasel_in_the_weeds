# Каталог `game/ui`

Этот каталог содержит элементы интерфейса, которые рисуются поверх игрового мира.

## Файлы

- `hud.py` — основной HUD игрока.

## Что показывает HUD

- здоровье;
- выносливость;
- монеты;
- `knowledge_shards`;
- хотбар;
- подсказки по кнопкам;
- подсказку о карте и крафте.

## Где используется

HUD вызывается из `game/scenes/game_scene.py` каждый кадр после отрисовки мира.
