# Каталог `game/world`

Этот каталог отвечает за представление мира, уровни и коллизии.

## Файлы

- `level.py` — загрузка уровня из JSON и валидация слоев.
- `tilemap.py` — работа с тайловой картой и отрисовка земли/препятствий.
- `collision.py` — проверка коллизий по карте и объектам.

## Как загружается уровень

1. `GameScene` вызывает `load_level(...)`.
2. Из JSON читаются:
   - имя уровня,
   - размер тайла,
   - точка спавна игрока,
   - слой земли,
   - слой препятствий,
   - список объектов.
3. На основе этого строится `TileMap` и система коллизий.

## Что важно помнить

- слои `ground` и `obstacles` должны быть одинакового размера;
- карта должна быть прямоугольной;
- объекты уровня лежат отдельно от тайлов и создаются через фабрику объектов.
