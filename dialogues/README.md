# Каталог `dialogues`

Здесь лежат JSON-файлы диалогов NPC.

## Что содержит один диалог

Обычно файл содержит:

- `id` — идентификатор диалога;
- `start` — стартовый узел;
- `nodes` — набор узлов.

Каждый узел может содержать:

- `speaker` — кто говорит;
- `text` — текст реплики;
- `next` — следующий узел;
- `choices` — список ответов игрока;
- `rewards` — награды за вход в узел.

## Награды

Сейчас узел может выдавать:

- `items`
- `coins`
- `knowledge_shards`
- `flags`

## Важное поведение

Повторные награды не должны выдаваться бесконечно. Для этого у игрока хранится информация о том, какой NPC и какой узел уже выдал награду.

## Как добавить новый диалог

1. Создать новый JSON в этом каталоге.
2. Указать его в `npc_object`.
3. Проверить, что все `next` и ссылки на узлы корректны.
