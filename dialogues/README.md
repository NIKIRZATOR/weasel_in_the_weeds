# Каталог `dialogues`

Здесь лежат JSON-файлы игровых диалогов.

## Базовая структура

Обычно файл содержит:

- `id` - идентификатор диалога
- `start` - стартовый узел
- `nodes` - список узлов диалога

Каждый узел может включать:

- `id` - идентификатор узла
- `speaker` - кто говорит: игрок или NPC
- `portrait_path` - необязательный путь к портрету
- `text` - текст реплики
- `next` - следующий узел
- `choices` - варианты ответа
- `rewards` - награды за вход в узел

## Награды

Сейчас диалоги поддерживают:

- `items`
- `coins`
- `knowledge_shards`
- `flags`

Награды не должны выдаваться бесконечно. Для этого игра хранит, какие узлы уже выдавали награду конкретному NPC.

## Как добавить новый диалог

1. Создать новый JSON-файл в этой папке.
2. Указать его в свойствах NPC.
3. Проверить переходы `next` и `choices`.
4. При необходимости задать портрет через `portrait_path`.
